Nomenclatura de ramas

SDA_xx_<d,p>

Donde: xx: indentificador de la rama: 	st:Singleton
				      	vn: vista nueva
					mn: modelo nuevo
					td: test drive
					p5: punto 5

	<d,p>: una letra que identifica si la rama es de desarrollo, o de prueba. Las ramas de pruebas hacen merge sobre las ramas de desarrollo
